# Life Organizer App

## Goal

The goal of this application is to help users organize their life using activities and reminders.

Users can create labels such as:

- Fitness
- Work
- Food
- Finance
- Personal

Inside each label users create activities.

Example:

Label: Fitness  
Activity: Gym  
Schedule: Every Tuesday at 18:00  
Notification: 30 minutes before

The application notifies users before activities.

---

## Core Features

### Labels

Users can create labels to categorize activities.

Example labels:

- Fitness
- Work
- Finance
- Health
- Personal

Each label can include:

- name
- color
- notes

---

### Activities

Activities belong to a label.

Each activity can contain:

- title
- notes
- schedule
- notification preference

Example:

Activity: Go to gym  
Label: Fitness  
Schedule: Weekly Tuesday 18:00  
Notification: 30 minutes before

---

### Scheduling

Activities support recurring schedules.

Supported schedules:

- daily
- weekly
- monthly
- custom

Users can also set notification timing.

Examples:

- 10 minutes before
- 30 minutes before
- 1 hour before

---

### Notifications

The main feature of the application is reminders.

Users will receive notifications when an activity time approaches.

Notifications should:

- trigger reliably
- work offline
- support recurring activities

---

### Views

The application provides multiple views:

Dashboard  
Calendar  
Timeline  
Label view  
Activity detail

---

## Design Philosophy

The application should be:

- extremely simple
- fast
- minimal
- easy to use
- clean UI
