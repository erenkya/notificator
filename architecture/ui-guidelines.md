# UI Guidelines

## Design Goals

The UI should be:

minimal  
clean  
light colored  
easy to understand  
extremely user friendly

The design should feel similar to:

Apple Reminders  
Notion  
Linear

---

## Colors

Primary: #6366F1

Background: #F8FAFC

Text Primary: #0F172A

Secondary Text: #64748B

Accent: #22C55E

---

## Spacing System

Use an 8pt spacing grid.

Examples:

8px  
16px  
24px  
32px

---

## Typography

Use a clean sans-serif font.

Hierarchy:

Large titles  
Section headers  
Body text  
Secondary text

---

## Components

Common UI components:

Label cards  
Activity cards  
Floating add button  
Timeline list  
Calendar view

---

## Interaction Design

Interactions should be:

fast  
smooth  
minimal taps

Users should be able to:

create a label quickly  
create an activity quickly  
see upcoming activities immediately
